First, obtain the latest version number of Apollo Android from [the Releases page](https://github.com/apollographql/apollo-android/releases).

Then in your app's Gradle file, apply the `com.apollographql.apollo3` plugin using one of the following methods (replace `x.y.z` with the desired version number):

**Using the [plugins DSL](https://docs.gradle.org/current/userguide/plugins.html#sec:plugins_block)**

<MultiCodeBlock>

```kotlin:title=build.gradle
plugins {
  // ...
  id("com.apollographql.apollo3").version("x.y.z")
}
```

```groovy:title=build.gradle
plugins {
  // ...
  id("com.apollographql.apollo3").version("x.y.z")
}
```

</MultiCodeBlock>

**Using the [legacy syntax](https://docs.gradle.org/current/userguide/plugins.html#sec:old_plugin_application)**

<MultiCodeBlock>

```kotlin:title=build.gradle
buildscript {
  // ...
  classpath("com.apollographql.apollo3:apollo-gradle-plugin:x.y.z")
}

apply(plugin = "com.apollographql.apollo3")
```

```groovy:title=build.gradle
buildscript {
  // ...
  classpath("com.apollographql.apollo3:apollo-gradle-plugin:x.y.z")
}

apply plugin: "com.apollographql.apollo3"
```

</MultiCodeBlock>

The plugin is hosted on both the Gradle plugin portal and Maven Central.

By default, the plugin generates models in the root package name. You can configure this in the `apollo {}` block:

```kotlin:title=build.gradle
apollo {
  packageName.set("com.example")
}
```
