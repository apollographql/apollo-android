In your app Gradle file, apply the `com.apollographql.apollo3` plugin:

Using the [plugins DSL](https://docs.gradle.org/current/userguide/plugins.html#sec:plugins_block):

<MultiCodeBlock>

```kotlin
plugins {
  // ...
  id("com.apollographql.apollo3").version("x.y.z")
}
```

```groovy
plugins {
  // ...
  id("com.apollographql.apollo3").version("x.y.z")
}
```

</MultiCodeBlock>

Or using the [legacy syntax](https://docs.gradle.org/current/userguide/plugins.html#sec:old_plugin_application):

<MultiCodeBlock>

```kotlin
buildscript {
  // ...
  classpath("com.apollographql.apollo3:apollo-gradle-plugin:x.y.z")
}

apply(plugin = "com.apollographql.apollo3")
```

```groovy
buildscript {
  // ...
  classpath("com.apollographql.apollo3:apollo-gradle-plugin:x.y.z")
}

apply plugin: "com.apollographql.apollo3"
```

</MultiCodeBlock>

The plugin is hosted on the Gradle plugin portal and Maven Central.
